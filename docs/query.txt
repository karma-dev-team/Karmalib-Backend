Query-запросы для форума
ПолучитьПостыПоКатегории: Возвращает посты по выбранной категории.
ПолучитьПостПоID: Возвращает данные поста по его ID.
ПолучитьКомментарииПоПосту: Возвращает все комментарии для указанного поста.
ПолучитьПостыПоТегу: Возвращает все посты, помеченные определенным тегом.
ПолучитьПопулярныеПосты: Возвращает список самых популярных постов (например, по лайкам или просмотрам).
ПолучитьЗакрепленныеПосты: Возвращает все закрепленные посты.
ПолучитьТегиФорума: Возвращает все доступные теги на форуме.
ПолучитьПодпискиПользователя: Возвращает список подписок пользователя.
ПолучитьИсториюИзмененийПоста: Возвращает историю изменений для конкретного поста.
ПолучитьИсториюИзмененийКомментария: Возвращает историю правок для конкретного комментария.
ПолучитьРеакцииНаКомментарий: Возвращает реакции на комментарий (например, лайки или эмодзи).
ПолучитьПомеченныеПосты: Возвращает список постов, отмеченных как нарушающие правила.
ПолучитьСтатистикуАктивностиПоПостам: Возвращает данные по количеству просмотров, лайков и комментариев для каждого поста.
ПолучитьИсториюПодписокПользователя: Показывает, когда и на какие посты или категории подписывался пользователь.
ПолучитьПопулярныеТеги: Возвращает теги, наиболее часто используемые на форуме.
ПолучитьОтветыНаКомментарий: Возвращает список ответов на конкретный комментарий.
ПолучитьПоследниеИзмененияФорума: Возвращает историю последних действий (например, созданных постов) на форуме.

ПолучитьИсториюАктивности: Возвращает историю действий пользователя в системе (например, комментирование, лайки).

ПолучитьЛичныеСообщения: Возвращает список личных сообщений пользователя.
ПолучитьИсториюИзмененийПрофиля: Возвращает историю изменений профиля, например, смены имени или аватара.
ПолучитьИсториюУведомлений: Возвращает архив уведомлений пользователя, включая просмотренные и устаревшие.
ПолучитьПараметрыСеансовПользователя: Возвращает данные по активным и прошедшим сеансам пользователя (например, устройства, IP-адреса).
ПолучитьСвязанныеСообщества: Показывает все сообщества или группы, в которых пользователь активно участвует.
ПолучитьИсториюСистемныхСообщений: Возвращает список системных сообщений, отправленных пользователю (например, предупреждения).
ПолучитьНастройкиУведомлений: Возвращает текущие настройки уведомлений, чтобы пользователь мог контролировать, какие уведомления он получает.
